## 📌 Overview
This is a cybersecurity project demonstrating how ransomware-like behavior works. The project includes:
- **Encryption & Decryption** scripts (`encrypt.py`, `decrypt.py`)
- **Key management** (`thekey.key`)
- **Simulation of executable files** (`file.exe`, `metro.exe`, etc.)
- **Security Analysis & Ethical Hacking Research**

## 🛠️ Setup & Usage

### 🔹 Prerequisites
Ensure you have:
- Python 3.x installed
- Required dependencies (install using `pip`)

### 🔹 Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/your-username/ransom-ware.git
   cd ransom-ware
   ```
2. Create and activate a virtual environment:
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

### 🔹 Running the Scripts
- **Encryption:**  
  ```sh
  python villain.py
  ```
- **Decryption:**  
  ```sh
  python decrypt.py
  ```
